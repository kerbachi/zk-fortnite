# Changelog

## 0.6.0 — Interactive greybox demo

- Browser demo under `demo/`: stub OAuth → username → mode select.
- Offline (and Online Singleplayer) Three.js arena: move, shoot, build, storm, bots.

## 0.5.1 — Dedicated repository

- Project hosted at https://github.com/kerbachi/zk-fortnite

## 0.5.0 — Online Singleplayer vs Multiplayer

- After **Play Online**, player chooses **Singleplayer** or **Multiplayer**.
- Friend invites by username exist **only** in Online Multiplayer.
- Online Singleplayer is solo queue with invite UI hidden.

## 0.4.0 — Username creation + friend invites

- First OAuth login requires a unique player username.
- Online invites by username into a party (refined in 0.5.0).

## 0.3.0 — Mode select + social sign-in

- Offline or Online before each match; Google / Microsoft / Apple OAuth.

## 0.2.0 — Online-first direction

- Internet multiplayer draft (superseded).

## 0.1.0 — Specification

- Initial offline-first draft (git history).
